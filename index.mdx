## What is Literacy Platform?

Literacy Platform is a full-stack web application built for managing children's literacy education. It provides a structured workflow from student registration through payment approval, initial assessment, and daily skill-based learning tasks — all controlled by an admin team.

## How it works

The platform operates as an admin-controlled lifecycle with four distinct phases:

<Steps>
  <Step title="Registration & Payment">
    A parent registers their child and submits payment via transaction ID or receipt upload. The account enters `pending_payment` status.
  </Step>
  <Step title="Admin Approval">
    An admin reviews the payment, approves or rejects it, then creates login credentials for the student. The account moves to `approved_pending_login`.
  </Step>
  <Step title="Initial Assessment">
    The student completes a multi-skill assessment (reading, listening, writing, speaking). The admin reviews responses and assigns a literacy level.
  </Step>
  <Step title="Daily Learning">
    The student receives daily skill tasks matched to their literacy level. Completed tasks earn Reward Points (RP), tracked in a ledger.
  </Step>
</Steps>

## Key features

<CardGroup cols={2}>
  <Card title="Four literacy levels" icon="layer-group">
    Students are placed at foundational, functional, transitional, or advanced based on their assessment results
  </Card>
  <Card title="Multi-skill assessments" icon="pen-to-square">
    Covers all four literacy domains: reading, listening, writing, and speaking
  </Card>
  <Card title="Reward Points system" icon="star">
    Students earn RP for completing daily tasks — every event is recorded in an immutable ledger
  </Card>
  <Card title="Secure file uploads" icon="cloud-arrow-up">
    Receipts and submission artifacts are stored in Cloudflare R2 via presigned URLs — never exposed through the app server
  </Card>
  <Card title="JWT authentication" icon="lock">
    Separate admin and student tokens with role claims — middleware enforces access on every protected route
  </Card>
  <Card title="Full audit log" icon="clock-rotate-left">
    Every admin action (payment approval, level assignment, content creation) is recorded in an append-only audit log
  </Card>
</CardGroup>

## Tech stack

| Layer | Technology |
|---|---|
| Framework | Next.js 16 (App Router) |
| Database | PostgreSQL via Prisma |
| File storage | Cloudflare R2 |
| Background jobs | Cloudflare Workers |
| Auth | JWT (HS256) via `jsonwebtoken` |
| Validation | Zod |
| Styling | Tailwind CSS v4 |

<Note>
  Literacy Platform is a private SaaS application — it is not published to npm and has no public SDK. All integrations happen via its REST API.
</Note>
